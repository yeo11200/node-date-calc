# 자동 적으로 시간 더해주는 함수 

### dateAddDel
```js
const jsAdd = require('js-date-calc');

// first : 시간 데이터
// second : 시간
// third : y, m, d
jsAdd.dateAddDel('', '', '');

```


### dateAddDelYmsHms
```js

/**
 * interval is kind 
 * year, month, week, day, hour, minute, second
 * */
dateAddDelYmsHms('date Start', 'interval', 'number')
```
